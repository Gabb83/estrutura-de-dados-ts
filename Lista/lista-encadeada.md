# Lista Encadeada

## Classe No 
A classe `No` representa um nó da lista encadeada. Em alusão, é como se ela fosse um vagão de um trem, onde cada vagão armazena um valor e uma referência para o próximo vagão.

O método construtor recebe um valor, inicializando, enquanto o ponteiro para o próximo nó é definido como `null`, indicando que, inicialmente, o nó não aponta para nenhum outro nó.

## Classe ListaEncadeada
A classe `ListaEncadeada` representa a estrutura da lista encadeada em si. Ela possui um atributo `head`, que é o ponteiro para o primeiro nó da lista. O método construtor inicializa `head` como `null`, indicando que a lista está vazia.
